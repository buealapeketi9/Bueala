# Loan Prediction using AI/ML

## Project Overview
This project predicts whether a loan application will be approved or not using Machine Learning algorithms.

## Features
- Loan Eligibility Prediction
- User-Friendly Interface
- Machine Learning Model
- Fast Prediction

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML
- CSS
